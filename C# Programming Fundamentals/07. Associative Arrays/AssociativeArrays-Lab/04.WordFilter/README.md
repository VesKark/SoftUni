# Word Filter
  
Read an **array of strings** and take only **words, whose length is even**. Print each word on a new line.

## Sample Input and Output  
    
| **Input** | **Output** |  
| :--- | :--- | 
| kiwi orange banana apple | kiwi<br> orange<br> banana |
| pizza cake pasta chips | cake |
