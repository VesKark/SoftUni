# Print Even Numbers
  
Create a program that:
- **Reads** an array of **integers** and **adds them to a queue**.
- **Prints** the **even numbers** separated by ", ".

## Sample Input and Output  
    
| **Input** | **Output** |  
| :--- | :--- | 
| 1 2 3 4 5 6 | 2, 4, 6 |
| 11 13 18 95 2 112 81 46 | 18, 2, 112, 46 |
